DATA SYNC PROCESS

(For Domestic Suppliers in GDSN)

In data sync the correct department preferences need to be selected. To create items within Data Sync, click on "New Items" under the "Data Sync Tasks" header in RL\OIF.

Items can be created by mass or single.

Single Item Create

  • Click on GTIN
  • Ensure that all information in Green is correct
  • Click Create

Mass Item Create

  • Check the box in front of the Product Description (multiple GTINs)
  • Download the "Item Creation Spreadsheet"
  • Proofing by Buyer, planner and RM
  • Import into Retail Link\ OIF\Create Item\Import completed Item Creation Spreadsheet

If information is not correct, go back into Data sync and place a Check in the box in front of the GTINs that had incorrect information.

Click on "Send back to Supplier for review".
