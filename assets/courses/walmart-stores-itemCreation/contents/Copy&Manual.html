COPY FUNCTION & MANUAL PROCESS

Option 1: Copy Function

Copies the provided item's information into the fields on the resulting single creation page allowing any necessary adjustments on the new Item. To copy:
  • Enter an existing Item Number
  • Click on Next button

Option 2: Manual Process

Requires information to be manually keyed into the fields on the resulting single creation page to create the item number. To utilize the manual process:
  • Select Department
  • Select Item Type
  • Click on Next button
