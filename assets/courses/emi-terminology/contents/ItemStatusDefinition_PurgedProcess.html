ITEM STATUS DEFINITIONS

Purged

The purge process is used to clean up the item file. After the item is purged, it cannot be scanned or ordered at store level. When the item meets the criteria of the Deletion or Inactivation process the system will determine if there has been activity on an item.

  • No Activity - If there has not been any activity the item will be purged from the system and it will no longer be visible in OIF, HOST, or store systems. However, queries can still be pulled from Retail Link to gather sales data

  • Activity - If the item had activities in the last 30 days it will remain "on file" in OIF, HOST, and store systems. The system looks at the item every week and purges it when there has been no activity for that week
