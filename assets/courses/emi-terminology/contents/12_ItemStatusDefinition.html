Item Status Definitions

The status of an item will determine how it can be used by merchandising and replenishment teams besides when the item can be eliminated from WM system

  • Active - An item in active status is available for the buying office and replenishment systems to place orders. It also allows a store to receive the product.

  • Deleted - If an item is changed to deleted status, replenishment is shut off and stores will be unable to receive orders. After 90 days of deletion status, the system will review it for activity and purge if it meets certain qualifications defined in the purge process.

  • Inactive - If an item is changed to inactive status, replenishment is shut off and stores will be unable to receive orders. The system will not try to purge an inactive item until 13 months after the status change was applied. This status is used on most seasonal items so the item number can be retained for warehouse history movement

  • CWO Flag - The Cancel When Out (CWO) flag sets up an automatic deletion process on warehouse type items. When the CWO flag is set to 'Y', the system will stop the inbound orders to the distribution centers (DCs) while still allowing the stores to pull from their appropriate DC. When the quantities are cleared from all the warehouse slots the item will automatically flip to deleted status

  • Expiration Date - The expiration date sets a specific future date in which the item is inactivated and deleted. When the expiration date is reached, the system will put the item into "Inactive" status for 90 days and then change to "Deleted" status

Purged

The purge process is used to clean up the item file. After the item is purged, it cannot be scanned or ordered at store level. When the item meets the criteria of the Deletion or Inactivation process the system will determine if there has been activity on an item.

  • No Activity - If there has not been any activity the item will be purged from the system and it will no longer be visible in OIF, HOST, or store systems. However, queries can still be pulled from Retail Link to gather sales data

  • Activity - If the item had activities in the last 30 days it will remain "on file" in OIF, HOST, and store systems. The system looks at the item every week and purges it when there has been no activity for that week
